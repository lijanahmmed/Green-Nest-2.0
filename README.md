# Green Nest

**Green Nest** is a modern and eco-friendly web application dedicated to plant lovers. It’s designed to help users explore, learn, and buy indoor and outdoor plants while promoting sustainable living. Built with Next.js and Firebase, this project focuses on a clean UI, smooth navigation, and user-friendly experience.

---

## Features

- **Plant Care Tips** – Get expert advice on watering, sunlight, and fertilization.
- **Meet Our Green Experts** – Learn from plant specialists and get personalized consultation.
- **Authentication System** – Secure login, register, and password reset using Firebase Auth.
- **Add Plants** – Easily record new plants with details like title, photo, and description
- **Contact & Feedback** – Users can reach out through a simple contact form.
- **Responsive Design** – Works perfectly on mobile, tablet, and desktop devices.

---

## Technologies Used

**Frontend:**

- Next.js
- Tailwind CSS
- Daisyui

**Backend:**

- Node.js
- Express.js
- MongoDB

**Other Tools:**

- Firebase Authentication
- Router (next/navigation)
- Toastify & SweetAlert2 (for UI interactions)

## Clone the Repository

Use the following command to clone the project:

```bash
git clone https://github.com/lijanahmmed/Green-Nest-Client.git

cd Green-Nest-Client
```

### Install Dependencies

```bash
cd Green-Nest-Client
npm install

```

### Start Frontend

```bash
npm run dev
```
